## Flappy Bird Game
A simple Flappy Bird game built in Java using the `Swing` framework.

### How to Run Game

```bash
git clone https://github.com/ahmetyazlyyev5/FlappyBird.git
cd FlappyBird
javac -d bin src/*.java
java -cp bin App
```
### Controls
Spacebar: fly / restart the game <br />
B: toggle bird hitbox
